Sojourn Standard

A mobile app for planning and managing your vacations and excursions

About the App

Sojourn Standard helps users plan, track, and manage their vacations with ease. Users can add vacations, assign excursions to those vacations, and receive scheduled alerts for start dates, end dates, and excursions.

Features

  • Create and manage vacation entries
  • Add excursions linked to specific vacations
  • Receive timely alerts via notifications
  • Search for vacations or excursions using keywords
  • Share vacation details with others

Technology Stack

  • Android Studio
  • Java
  • Room (SQLite ORM)
  • Material Design Components
